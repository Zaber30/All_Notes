A **load balancer is a server or software that distributes incoming user requests across multiple servers so that no single server becomes overloaded.** It acts as a middle layer between users and backend servers.

---

### Easy Explanation

Imagine a restaurant with many customers.

- The **customers** = users
- The **waiter/receptionist** = load balancer
- The **chefs** = servers

The waiter sends each order to an available chef so that one chef does not get all the work.
